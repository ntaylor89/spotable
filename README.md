# Spotable

__Work in progress__

Build better playlists with data from the [Spotify API](https://developer.spotify.com/web-api/).

## Installation
```
git clone https://github.com/ntaylor89/spotable.git
cd spotable
npm i
```

## Usage
```
npm start
open localhost:3000
```
